# Boxing Performance Tracker

## Overview

This project is a Python application designed to track and analyse boxing training performance over time. Training data is stored in an Excel spreadsheet and analysed using Python to monitor key performance metrics, identify trends and visualise progress. The project demonstrates how data analysis can be applied to sports performance using real training data.

## Objectives

- Record boxing training sessions in a structured format.
- Analyse changes in training volume over time.
- Track body weight alongside training performance.
- Calculate key performance metrics.
- Visualise trends to support training analysis.

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Excel

## Project Features

- Import training data from Excel
- Weight tracking
- Total rounds analysis
- Sparring round analysis
- Training duration tracking
- Rounds per Hour calculation
- Average training statistics
- Personal best session identification
- Data visualisation using line charts
